# Map metadata

Metadata can be provided for map defintions.
Metadata includes:
* WMDL version
* Map title 
* Arrangement of the evolution axis (Names, # of segments, range for every segment, ...)
* Title for both axis
* [Display-hints](../tokens/display-hint.md)
* Creation date
* Author(s)
* Generator (e.g. Onlinewardleymaps)
* Revision
